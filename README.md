# Панель управления ⛅ информационными картами Freelancer (2003) для Lizerium модификации (универсальна)

## Зачем я это написал?

- У меня при попытке создать `общий` `алгоритм` `генерации` `карт` описаний и `названий` игры сформировалась папка из `сотен` вариантов этих карт, многие файлы имели одинаковые названия в разных папках и предназначались для разных ситуаций `сравнения` или `уточнения`
- Поэтому я решил сделать центральный `HUB` - который расшаривал бы интерфейс взаимодействия с файлами которые являлись `релизным` вариантом текстового представления описаний и названий

## Что может ❓

- Кладём приложение в `корень` папки и имеем возможность:
  - открыть точный файл для конкретной библиотке dll
  - скомпилировать результирующую dll утилитой frc.exe

## Что нужно сделать ❓

- Сделать пути до `файлов` и `exe` регулируемыми через конфигурационный файл и настройки
  - Сейчас пути буквально зашиты в `код` приложения но вынесены в отдельный интерфейс где мы может менять их
